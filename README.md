Gabi's Store

About
An application for posting and reviewing products.

Setup
1) Clone this repository.
2) Run the following commands in the root of the project to set up the application
 a) bundle install
 b) rails db:create
 c) rails db:schema:load
 d) rails s
 e) rails db:seed
